# Berserk Project

## Project Information
This project is inspired by the world and characters of Berserk, particularly focusing on Guts, the Black Swordsman. The aim is to explore themes of struggle, survival, and the relentless pursuit of one's goals in a dark fantasy setting.

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/sandrotonal/berserk.git
   ```
2. Navigate into the project directory:
   ```bash
   cd berserk
   ```
3. Install required dependencies:
   ```bash
   <installation-command>
   ```

## Project Structure
- `src/` - Contains all source files.
- `tests/` - Contains unit and integration tests.
- `docs/` - Documentation for the project.
- `README.md` - Project overview and instructions.
